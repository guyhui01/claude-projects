## User Story
En tant qu'**acheteur B2B**,
Je veux **voir les frais de livraison calculés automatiquement** selon mon adresse et le poids,
Afin de **connaître le coût total avant validation**.

## Critères d'acceptation

**Scénario 1 — Calcul automatique des frais**
```gherkin
Given je suis à l'étape livraison avec mon adresse saisie
When  le système calcule les frais
Then  le montant des frais de port HT s'affiche avant validation
And   le total commande est recalculé avec les frais inclus
```

**Scénario 2 — Franchise de port**
```gherkin
Given ma commande dépasse le seuil de franchise (ex : 500€ HT)
When  je suis à l'étape livraison
Then  "Livraison offerte" s'affiche
```

## Dépendance
