## User Story
En tant que **commercial Company-Test**,
Je veux **exporter les commandes au format CSV**,
Afin de **les traiter manuellement en attendant l'intégration ERP**.

## Critères d'acceptation

**Scénario 1 — Export CSV**
```gherkin
Given je suis sur le tableau de bord commandes
When  je clique sur "Exporter CSV" (avec filtre période optionnel)
Then  un fichier CSV est téléchargé contenant :
      · N° commande · date · client · SIRET
      · Produits · quantités · prix unitaires HT
      · Total HT · statut · N° BC
```

**Scénario 2 — Export filtré par période**
```gherkin
Given je filtre les commandes du mois en cours
When  je clique sur "Exporter CSV"
Then  seules les commandes de la période filtrée sont dans le CSV
```

## Note
Cette US est temporaire — sera remplacée par l'intégration ERP/SAP en phase 2.
