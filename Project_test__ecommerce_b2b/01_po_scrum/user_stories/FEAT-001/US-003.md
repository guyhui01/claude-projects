## User Story
En tant qu'**acheteur B2B**,
Je veux **filtrer les produits** par référence, type de câble et conditionnement,
Afin de **réduire mon temps de recherche**.

## Critères d'acceptation

**Scénario 1 — Filtre actif**
```gherkin
Given je suis sur la page catalogue
When  j'applique le filtre "Conditionnement : bobine 500m"
Then  seuls les produits correspondants sont affichés
And   le filtre actif est visible avec une croix de suppression
```

**Scénario 2 — Réinitialisation des filtres**
```gherkin
Given j'ai appliqué plusieurs filtres
When  je clique sur "Réinitialiser les filtres"
Then  tous les filtres sont supprimés
And   le catalogue complet s'affiche
```
