## User Story
En tant qu'**acheteur B2B**,
Je veux **consulter la fiche produit complète** (référence, specs, tarif B2B, stock),
Afin de **prendre ma décision d'achat** sans contacter un commercial.

## Critères d'acceptation

**Scénario 1 — Affichage fiche produit connecté**
```gherkin
Given je suis connecté en tant qu'acheteur B2B validé
When  j'accède à la fiche d'un câble FTTH
Then  je vois : référence · description · specs techniques
And   je vois : prix HT B2B · unité de vente · délai livraison · disponibilité stock
```

**Scénario 2 — Prix masqué non connecté**
```gherkin
Given je ne suis pas connecté
When  j'accède à une fiche produit
Then  le prix est masqué
And   le message "Connectez-vous pour voir vos tarifs" s'affiche
```

**Scénario 3 — Ajout au panier depuis la fiche**
```gherkin
Given je suis connecté et le produit est en stock
When  je clique sur "Ajouter au panier"
Then  le produit est ajouté avec la quantité saisie
And   un message de confirmation s'affiche
```
