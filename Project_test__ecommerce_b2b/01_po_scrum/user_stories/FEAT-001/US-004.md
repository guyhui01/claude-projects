## User Story
En tant qu'**acheteur B2B**,
Je veux **télécharger la fiche technique PDF** depuis la fiche produit,
Afin de **la partager avec mon équipe technique** avant commande.

## Critères d'acceptation

**Scénario 1 — Téléchargement disponible**
```gherkin
Given je suis sur une fiche produit avec PDF associé
When  je clique sur "Télécharger la fiche technique"
Then  le PDF se télécharge sans redirection ni connexion requise
```

**Scénario 2 — PDF absent**
```gherkin
Given je suis sur une fiche produit sans PDF associé
When  je consulte la page
Then  le bouton de téléchargement n'est pas affiché
```
