## User Story
En tant qu'**acheteur B2B**,
Je veux **naviguer dans le catalogue par catégorie** (Câbles FTTH / Boîtiers fibres),
Afin de **trouver rapidement les produits** correspondant à mon besoin.

## Critères d'acceptation

**Scénario 1 — Navigation par catégorie**
```gherkin
Given je suis sur la page d'accueil du catalogue
When  je clique sur la catégorie "Câbles FTTH"
Then  je vois uniquement les produits de cette catégorie
And   le nombre de résultats est affiché
```

**Scénario 2 — Breadcrumb de navigation**
```gherkin
Given je suis dans une sous-catégorie
When  je consulte le fil d'Ariane
Then  le chemin complet est visible
And   chaque niveau est cliquable
```
