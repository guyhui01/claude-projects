# US-009 — Ajout produits au panier

| Champ | Valeur |
|-------|--------|
| Feature | FEAT-003 — Panier & processus de commande |
| Epic | EPIC-001 |
| MoSCoW | Must Have |
| Story Points | 3 |
| Sprint | Sprint 2 |
| Statut | À faire |

## User Story
En tant qu'**acheteur B2B**,
Je veux **ajouter des produits au panier** en précisant la quantité,
Afin de **préparer ma commande en une session**.

## Critères d'acceptation

**Scénario 1 — Ajout au panier réussi**
```gherkin
Given je suis sur une fiche produit
When  je saisis une quantité valide et clique "Ajouter au panier"
Then  le produit apparaît dans le panier avec la quantité et le sous-total HT
```

**Scénario 2 — Quantité minimum non respectée**
```gherkin
Given la quantité minimum est de 10 unités
When  je saisis une quantité inférieure à 10
Then  le message "Quantité minimum : 10" s'affiche
And   l'ajout au panier est bloqué
```

**Scénario 3 — Modification quantité dans le panier**
```gherkin
Given un produit est dans mon panier
When  je modifie la quantité
Then  le sous-total et le total HT se mettent à jour automatiquement
```
