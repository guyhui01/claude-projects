## User Story
En tant qu'**acheteur B2B**,
Je veux **saisir mon numéro de bon de commande interne** lors de la validation,
Afin de **faciliter le rapprochement comptable** côté client.

## Critères d'acceptation

**Scénario 1 — Champ BC visible et optionnel**
```gherkin
Given je suis à l'étape de validation de commande
When  je consulte le formulaire de validation
Then  un champ "N° Bon de commande (optionnel)" est présent
And   la commande peut être validée sans renseigner ce champ
```

**Scénario 2 — N° BC reporté sur les documents**
```gherkin
Given j'ai saisi un numéro de BC "BC-2026-00123"
When  ma commande est confirmée
Then  ce numéro apparaît sur l'email de confirmation
And   ce numéro apparaît sur la facture PDF
```
