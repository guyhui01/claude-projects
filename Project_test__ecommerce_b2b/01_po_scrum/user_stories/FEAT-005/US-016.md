## User Story
En tant qu'**acheteur B2B**,
Je veux **payer par carte bancaire** via un module sécurisé (3D Secure),
Afin de **finaliser ma commande immédiatement**.

## Critères d'acceptation

**Scénario 1 — Paiement CB réussi**
```gherkin
Given je suis à l'étape de paiement
When  je saisis mes informations CB et valide (3D Secure OK)
Then  ma commande est confirmée
And   je reçois l'email de confirmation avec la facture PDF
```

**Scénario 2 — Paiement refusé**
```gherkin
Given ma carte est refusée par la banque
When  je valide le paiement
Then  un message "Paiement refusé — veuillez réessayer ou choisir un autre moyen" s'affiche
And   ma commande n'est pas enregistrée
```
