## User Story
En tant que **nouvel acheteur B2B**,
Je veux **créer un compte professionnel** en renseignant ma raison sociale, SIRET et coordonnées,
Afin d'**accéder au catalogue et aux tarifs B2B**.

## Critères d'acceptation

**Scénario 1 — Création de compte réussie**
```gherkin
Given je suis sur la page d'inscription B2B
When  je renseigne tous les champs obligatoires et valide
Then  mon compte est créé avec statut "En attente de validation"
And   je reçois un email de confirmation de demande
```

**Scénario 2 — SIRET invalide (format)**
```gherkin
Given je saisis un SIRET qui ne contient pas exactement 14 chiffres
When  je tente de valider le formulaire
Then  un message d'erreur s'affiche sur le champ SIRET
And   la soumission est bloquée
```

**Scénario 3 — Email déjà utilisé**
```gherkin
Given je saisis un email déjà enregistré
When  je soumets le formulaire
Then  le message "Un compte existe déjà avec cet email" s'affiche
```
