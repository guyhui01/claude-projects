## User Story
En tant qu'**acheteur B2B**,
Je veux **voir mes tarifs négociés spécifiques** sur les fiches produits,
Afin de **préparer mes commandes avec les bons prix**.

## Critères d'acceptation

**Scénario 1 — Tarif personnalisé affiché**
```gherkin
Given mon compte a des tarifs négociés configurés par Company-Test
When  je consulte une fiche produit concernée
Then  mon prix négocié HT s'affiche à la place du tarif standard
And   un label "Votre tarif négocié" est visible
```

**Scénario 2 — Pas de tarif négocié**
```gherkin
Given mon compte n'a pas de tarif négocié pour ce produit
When  je consulte la fiche produit
Then  le tarif standard B2B s'affiche
```

## Dépendance
⚠️ Grille des tarifs négociés à recevoir de la Direction Commerciale avant Sprint 4.
