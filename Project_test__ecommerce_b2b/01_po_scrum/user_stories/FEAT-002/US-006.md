## User Story
En tant qu'**acheteur B2B enregistré**,
Je veux **me connecter avec email + mot de passe**,
Afin d'**accéder à mon espace personnel et mes commandes**.

## Critères d'acceptation

**Scénario 1 — Connexion réussie**
```gherkin
Given mon compte est validé par Company-Test
When  je saisis mes identifiants corrects
Then  je suis redirigé vers mon tableau de bord B2B avec mes tarifs
```

**Scénario 2 — Compte en attente de validation**
```gherkin
Given mon compte est en attente de validation
When  je tente de me connecter
Then  le message "Votre compte est en cours de validation (délai 24h)" s'affiche
```

**Scénario 3 — Identifiants incorrects**
```gherkin
Given je saisis un mot de passe incorrect
When  je valide la connexion
Then  le message "Email ou mot de passe incorrect" s'affiche
And   le compte n'est pas bloqué avant 5 tentatives
```
