# Projet — Company-Test e-Commerce B2B Drupal

## Stack technique
- CMS : Drupal 10 + Drupal Commerce 2.x
- PSP : Stripe (à confirmer après spike Iteration 1)
- Email transactionnel : SendGrid
- Transporteur : API Chronopost / Colissimo
- Export : CSV manuel — pas d'intégration ERP/SAP en V1

## Contexte métier
- Client : Company-Test (fabricant câbles FTTH + boîtiers fibres)
- Marché : B2B France — installateurs, opérateurs, distributeurs
- Périmètre V1 : Câbles FTTH + Boîtiers fibres
- Hors périmètre V1 : Marketplace · B2C · connecteurs ERP/SAP (phase 2)

## PI en cours
- PI-2026-Q3 — 10 semaines — 4 Iterations + IP Sprint
- Vélocité équipe : ~18 pts/iteration
- Équipe : 4 dev Drupal · 1 PO · 1 SM

## Conventions projet
- Nommage livrables : MAJUSCULE-ID (ex: EPIC-001.md, FEAT-003.md, US-012.md)
- Nommage fichiers process : snake_case (ex: product_backlog.md)
- Langue : français
- Encodage : UTF-8
